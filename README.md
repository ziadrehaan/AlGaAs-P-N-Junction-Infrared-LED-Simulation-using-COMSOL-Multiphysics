# 💡 AlGaAs P–N Junction Infrared LED Simulation

A semiconductor device simulation project developed using **COMSOL Multiphysics 6.2**, focusing on the design and analysis of an **Al₀.₃Ga₀.₇As P–N Junction Infrared LED**.

The project investigates **carrier transport**, **radiative recombination**, and **electroluminescence** to evaluate the performance and efficiency of infrared light-emitting devices.

---

## 🚀 Project Overview

This simulation models an **AlGaAs-based infrared LED** using a **2D axisymmetric geometry**, providing accurate semiconductor analysis while reducing computational complexity.

The device was designed to study the electrical and optical behavior of an infrared LED under forward-bias conditions.

## ⚙️ Key Features

✅ Al₀.₃Ga₀.₇As semiconductor material

✅ Carrier transport analysis

✅ Radiative recombination modeling

✅ Electroluminescence evaluation

## 🔬 Semiconductor Parameters

The simulation includes manually defined semiconductor properties such as:

* ⚡ Bandgap Energy (Eg)
* 🔋 Electron Affinity (χ)
* 📈 Effective Density of States (Nc & Nv)
* 🚀 Electron Mobility
* 🔄 Hole Mobility
* ⏳ Carrier Lifetimes (τn & τp)

These parameters were configured to accurately represent **Al₀.₃Ga₀.₇As** within the Semiconductor Module.

## 📈 Results

The simulation successfully demonstrates:

* ❤️ Carrier injection across the P–N junction
* ⚡ Current density distribution
* 💡 Radiative recombination inside the active region
* 🌈 Infrared electroluminescence generation
* 📊 Device efficiency analysis

The highest recombination rate occurs near the active junction, confirming the expected operation of an AlGaAs infrared LED.
